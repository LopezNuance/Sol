# RFC-SOL-0001 v0.1.0-draft-freeze

This repository implements against the frozen RFC text produced in the design session. The full conversation draft is represented operationally by:

- README.md summary
- schemas/*.schema.json
- validator rule catalog in solval.validate
- corpus manifest generated by solval.make_corpus

The normative validation rule IDs V0-V5 are implemented as executable checks in the validator prototype. The exploded debug representation is used for implementation and corpus construction before a physical `.solnb` container is selected.
